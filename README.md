## Bookxpert AI Assignment

This repository contains the solution for the AI Developer Assignment given by Bookxpert Pvt Ltd.
It includes two tasks:

### Task 1 — Name Matching System

A Python-based system that:

Stores a list of names

Takes user input

Finds the closest matching name

Shows similarity scores

Displays a ranked list of similar names

### Task 2 — Local LLM Recipe Chatbot

A local AI-powered chatbot that:

Reads user ingredients

Suggests a relevant recipe

Uses a FastAPI backend

Returns responses in JSON format

How to Run the Project
1. Create Virtual Environment
python -m venv venv

2. Activate Environment

Windows:
```
.\venv\Scripts\Activate.ps1
```
